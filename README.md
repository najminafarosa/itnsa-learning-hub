# 🌐 ITNSA Learning Hub

Selamat datang di **ITNSA Learning Hub**! 🚀  
Repositori ini adalah pusat belajar mandiri dan tempat berbagi *resource* seputar **Information Technology Network Systems Administration (ITNSA)**. 

Tempat ini dibuat khusus untuk membagikan materi, panduan praktis, *source code* otomatisasi, serta kumpulan soal latihan untuk mengasah kemampuan di bidang administrasi jaringan dan sistem.

---

## 📂 Struktur Repositori

Agar belajar jadi lebih mudah, repositori ini dibagi menjadi beberapa bagian utama:

*   📂 **`docs/`** – Dokumentasi sistem, topologi jaringan, panduan konfigurasi, dan catatan teori penting.
*   📂 **`exercises/`** – Kumpulan soal latihan, studi kasus, lab mandiri, dan tantangan (challenge) untuk menguji *skill*.
*   📂 **`src/`** – *Source code* asli, termasuk skrip otomatisasi (Bash/Python), konfigurasi *Infrastructure as Code* (Ansible/Terraform), dan *file* konfigurasi server.

---

## 🎯 Fokus Pembelajaran

Di dalam *hub* ini, kita akan banyak mengeksplorasi topik-topik berikut:
1.  **Network Infrastructure:** Routing, Switching, VLAN, dan Network Security.
2.  **System Administration:** Setup & pengerasan (hardening) server Linux/Windows (DNS, Web, Mail, Storage).
3.  **Network Automation & Programmability:** Otomatisasi tugas jaringan menggunakan kode.
4.  **DevOps & Cloud Basic:** Implementasi Docker, CI/CD sederhana, dan pengelolaan infrastruktur modern.

---

## 🚀 Cara Menggunakan Repo Ini

1.  **Clone Repositori:**
    ```bash
    git clone https://github.com
    ```
2.  **Pilih Modul:** Masuk ke folder `docs/` untuk membaca panduan, atau langsung ke `exercises/` untuk mencoba tantangan lab.
3.  **Praktikkan:** Gunakan simulator seperti GNS3, Cisco Packet Tracer, atau VM (VirtualBox/Proxmox) untuk menyelesaikan soal latihan.
